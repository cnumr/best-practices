## Description

Events delegation avoids overloading browser memory by instantiating a single listener for several DOM (Document Object Model) elements.

Instead of assigning an event listener to several elements on the page, you can put a single event listener on a parent element of these objects. It is in this event handler that the triggering element will be determined using `event.target`.

More informations : [https://fr.javascript.info/event-delegation](https://fr.javascript.info/event-delegation)

## Exemple

Example script to highlight a table cell:

```javascript
<script type="text/javascript">
  let selectedTd;
  table.onclick = function(event) {
    let target = event.target;
    if (target.tagName != 'TD') return;
    highlight(target);
  };
  function highlight(td) {
    if (selectedTd) {
      selectedTd.classList.remove('highlight');
    }
    selectedTd = td;
    selectedTd.classList.add('highlight');
  }
</script>

```

Here we do not set an event listener on each cell but only on the table.
