## Descripción

Utilizar calificaciones CSS abreviadas para reducir el peso de la hoja de estilos.

## Ejemplo

No escriba lo siguiente:
```css
margin-top:1em; margin-right:0; margin-bottom:2em; margin-left:0.5em;
```
sino más bien:
```css
margin:1em 0 2em 0.5em;
```
