## Description

Utiliser les notations CSS abrégées pour réduire le poids de la feuille de styles.

## Exemple

Ne pas écrire :

```css
margin-top:1em; margin-right:0; margin-bottom:2em; margin-left:0.5em;
```

mais plutôt :

```css
margin:1em 0 2em 0.5em;
```
