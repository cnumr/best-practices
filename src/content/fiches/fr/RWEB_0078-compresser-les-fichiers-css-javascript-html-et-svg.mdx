## Description

Pour limiter l’utilisation de la bande passante et améliorer les temps de chargement, compressez les fichiers texte : CSS, JS, HTML et SVG. L’algorithme permet de compresser à la volée les ressources avant de les envoyer aux clients.

Plus récemment, `BROTLI` a été popularisé avec des performances accrues. Il est supporté par tous les navigateurs les plus répandus.

## Exemple

Pour utiliser `BROTLI` dans NGinx, il suffi t d’ajouter la configuration suivante :

```
# for compressing responses on-the-fly
load_module modules/ngx_http_brotli_filter_module.so;
# for serving pre-compressed files
load_module modules/ngx_http_brotli_static_module.so;
http {
    server {
    brotli on;
    #...
    }
}
```
